# Full-Stack Web Development Learning Repo 🚀

This repository documents my journey of learning full-stack web development using the **MERN-style stack** (React + Node.js) along with core web technologies: **HTML, CSS, and JavaScript**.

## 📚 What I'm Learning

- **Frontend:**
  - HTML5 for structure and semantics
  - CSS3 for styling, layout (Flexbox, Grid), and responsive design
  - React.js for building dynamic, component-based user interfaces
  - State management, hooks, and React Router

- **Backend:**
  - Node.js and Express.js for building RESTful APIs
  - Working with databases (MongoDB or SQL)
  - Authentication, authorization, and security basics

- **Full-Stack Integration:**
  - Connecting React frontend with Node.js backend
  - Consuming APIs with `fetch` / `axios`
  - Deploying full-stack applications

## 🛠 Technologies & Tools

- HTML5, CSS3, JavaScript (ES6+)
- React
- Node.js & Express
- Git & GitHub
- (Optional: MongoDB, Postman, VS Code, etc.)
